# Event Creation Requirements

## Required Fields for Creating an Event:

### 1. **Event Title** (Required)
- Text field, max 200 characters
- Example: "Tech Talk: AI in Web Development"

### 2. **Description** (Required) 
- Textarea, describe what the event is about
- Example: "Join us for an exciting discussion about AI tools for developers"

### 3. **Event Date** (Required)
- Date picker, must be today or future date
- Example: "2024-12-15"

### 4. **Start Time** (Required)
- Time picker in 24-hour format
- Example: "14:30" (2:30 PM)

### 5. **Maximum Registrations** (Required)
- Number field, 1-1000 participants
- Example: "50"

### Optional Fields:

### 6. **Venue** (Optional)
- Dropdown with available campus venues
- Can be left empty for online/TBD events

### 7. **Event Image** (Optional)
- Image upload (coming soon feature)

## Prerequisites Before Creating Events:

### You Need a Club Profile First:
- **Club Name**: Your club's official name
- **Description**: What your club does and its mission

## Current Flow:
1. **Sign up** as Club Representative
2. **Create Club Profile** (one-time setup)
3. **Create Events** (unlimited)

## After Event Creation:
- Events appear on your Club Dashboard
- Students can register for your events
- You can track registrations and manage attendees
- Post-event feedback collection (coming soon)

## Troubleshooting:
If you see "Create Club Profile" screen:
- This is normal for first-time club representatives
- Fill in your club name and description
- Click "Create Club Profile"
- You'll then be able to create events
