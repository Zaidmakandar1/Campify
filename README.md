# Welcome to Campify

## Project info

This repository contains the Campify web application.

## How to run the project locally

You can run the project locally using Node.js and npm. If you don't have Node.js installed, install it first (nvm or the official installer).

Basic steps:

```sh
# Clone the repository
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm install

# Start the development server
npm run dev
```

You can edit files in your preferred IDE and commit/push changes as usual.

## Technologies used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

Deploy the build output to your preferred static host or platform that supports Vite-built apps (Netlify, Vercel, GitHub Pages, etc.).

## Custom domains

Connect a custom domain using your hosting provider's settings for the deployed site. Consult your host's documentation for instructions.
